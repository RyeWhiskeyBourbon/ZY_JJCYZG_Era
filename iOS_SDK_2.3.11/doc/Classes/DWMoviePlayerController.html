response 形如：

Discussion

response 形如：

{ “result”: 1, “UPID”: “4109151396450044835”, “video”: { “status”: 0, “statusinfo”: “正常”, “title”: “步步高音乐手机广告”, “shareurl”: “”, “defaultquality”: 10 }, “qualities”: [{ “quality”: 10, “desp”: “清晰”, “copies”: [{ “playurl”: “http://bm2.33.play.bokecc.com/flvs/ca/Qxc52/uolaj09ZUD-10.mp4?t=1396457244&key=7A9DEB2C7BEFC4EFC4046CC666040054”, “priority”: 100 }, { “playurl”: “http://cm22.c110.play.bokecc.com/flvs/ca/Qxc52/uolaj09ZUD-10.mp4?t=1396457244&key=7A9DEB2C7BEFC4EFC4046CC666040054”, “priority”: 99 }] }] }

解析后的结果存放到 self.playUrls，形如：

{ “status”: 0, “statusinfo”: “正常”, “defaultquality”: 10 “qualityDescription”: [@“清晰”] “qualities”: [{ “quality”: 10, “desp”: “清晰”, “playurl”: “http://bm2.33.play.bokecc.com/flvs/ca/Qxc52/uolaj09ZUD-10.mp4?t=1396457244&key=7A9DEB2C7BEFC4EFC4046CC666040054&upid=4109151396450044835” }] }

解析原则： 1. 将 UPID 拼接到 playurl 的后面 2. 如果 qualities 中没有 defaultquality，则设置 defaultquality 改为 qualities 中最小的 quality。
